How to > Configure Facebook Insights

In this tutorial, you can find the instructions to import your Facebook Insights API key (access token), in order to configure your Facebook dashboard in Doutdes. Please perform the following steps in strict order, and do not forget to contact us for any problem.

Also remember that you can click over any image, to zoom it.

1) Sign up on developers.facebook.com and create a new Facebook app or use an existing app if you already have one.

2) Now go to Facebook Graph API Explorer. First select the application created, then select Get user access token.

3) A pop-up window will appear like below. Select read_insights and pages_show_list. Click on Get Access Token button to obtain a short-lived user access token.

4) Now, go to Access Token Tool. There you will see the access token for all the apps you have under your account. Click on Debug button.

5) In this page you will see that your access token will expire after an hour. So, click on Extend Access Token and you will get a long-lived access token. Copy the token (is needed for the next step).

6) Go again to Facebook Graph API Explorer. Paste the recently created long-lived token in the Access Token field. Now change the API endpoint with "/me/accounts" and click on Submit button. Now you can see the result with all pages information including the never expiring token. Copy that token like the image below.
