How to > Configure Facebook Insights

In this tutorial, you can find the instructions to import your Facebook Insights API key (access token), to configure your personal Facebook dashboard in DoUtDes. Recall to perform the following steps in strict order; you can also contact us for any problem. Also remember that you can click over any image, to zoom it.

1) Sign up on developers.facebook.com and create a new Facebook app (or use an existing app if you already have one):

2) Now go to the Facebook Graph API Explorer. Choose the app just created, then select Get user access token:

3) A pop-up window will appear, like below. Select read_insights and pages_show_list. Click on Get Access Token button to obtain a short-lived user access token:

4) Now, open the Access Token Tool. Here, you can see the access token for all the apps under your account. Click on the Debug button:

5) Observe that your access token expires in one hour. So, click on Extend Access Token to get a long-lived access token. Then, Copy your token (for the next step):

6) Come back to the Facebook Graph API Explorer. Paste the long-lived token (you just copied in step 5) into the Access Token field. Change the API endpoint with "/me/accounts" and click on Submit. A summary will be shown, including the never expiring token. Copy that token, as in the image:

7) Done! Now open DoUtDes, search for Preferences -> Configure data sources on the left menu, and click on Add new data source:

8) Select Facebook Insights as service type, then paste the never-expiring token. Conclude with Insert:

9) Good job! Now you have added your token in DoUtDes. Your Facebook Insights stats are now available as below:
