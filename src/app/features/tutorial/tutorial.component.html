Tutorial

1) Go to console.developer.google.com and create a new project:

2) From the dashboard click on Enable APIs and services:

3) From the list search for Google Analytics Reporting API and Analytics API and enable it:

4) Go back to the dashboard and click on Credential -> Create credential and choose Service account key:

5) Fill the form and choose JSON key type. Once filled save the JSON file on your PC:

6) Now go to DoUtDes platform and under preferences -> Configure data sources click on Add new data source:

7) Choose Google Analytics and upload your JSON file clicking on Insert:

8) Copy this service account e-mail:

9) Go to analytics.google.com and under Administration -> View click on User Management like the image below:

10) Click on + button and Add new user:

11) Paste the service account email and let read and analysis chosen:

12) Once finished, the Google Analytics charts are available on DoUtDes platform like the image below:
