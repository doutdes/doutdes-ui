How to > Configure website analytics

In this tutorial, you can find the instructions to import your Google Analytics API key (access token), in order to configure your website dashboard in Doutdes. Please perform the following steps in strict order, and do not forget to contact us for any problem.

Also remember that you can click over any image, to zoom it.

1) Go to console.developer.google.com and create a new project, assigning it a custom name:

2) Click on the Create button and, in the new page, go to Enable APIs and services:

3) In the list, search for Google Analytics Reporting API and Analytics API and enable them:

4) Now, go back to Dashboard in the side menu, search for Credentials -> Create credential and choose Service account key:

5) Fill the form for creating a new service account, and select the JSON key type. Once done, create the account and save the .json file on your PC:

6) Now, open DoUtDes, search for Preferences -> Configure data sources on the left menu, and click on Add new data source:

7) Choose Google Analytics and upload the .json file that you previously stored locally:

8) Almost done! A few steps more... let us copy the service account e-mail:

9) Open analytics.google.com, go to Administration -> View, and click on User Management:

10) Now, tap on the (+) button and then Add new user:

11) Paste here the service account email you copied before and enable the Read and analysis permissions:

12) Good job! Your website stats based on Google Analytics are now available on DoUtDes as below:
